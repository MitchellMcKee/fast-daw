Fast DAW
Fast DAW is a web app used for asynchronous playback of multiple audio files developed end-to-end by Mitchell McKee. Select an audio file from the dropdown and set how long you want to delay the start of that track. If you want to add your own audio, select an MP3 file from your computer and click the upload button. Repeat this process with as many tracks as you like. Once everything is set, hit start at the bottom of the screen and enjoy! You can adjust the volume of each individual track during playback too! Click here for a demo
